# Documentation [Install, configure, and use Starknet Start]

Use these docs to add Starknet Start to a React app, configure wallet and RPC
providers, call contracts, send transactions, and query Starknet network state.

## Setup

- [Getting Started](/docs/getting-started): install Starknet Start and configure `StarknetConfig`.
- [Migrating to Starknet Start](/docs/migrating-to-starknet-start): move from `starknet-react` or `@starknet-react/core`.
- [StarknetConfig](/docs/starknet-config): configure supported chains, providers, connectors, and explorers.
- [Wallets](/docs/wallets): configure recommended and injected wallet connectors.
- [RPC Providers](/docs/providers): choose public or hosted Starknet JSON-RPC providers.
- [Paymaster Providers](/docs/paymaster-providers): estimate and send gasless transactions.
- [Explorers](/docs/explorers): generate explorer links for user-facing transaction and account views.

## Hook reference

- [Wallet actions](/docs/hooks/use-connect): connect accounts, sign messages, switch chains, and send transactions.
- [Smart contracts](/docs/hooks/use-contract): instantiate contracts, read data, and deploy contracts.
- [RPC methods](/docs/hooks/use-block): query blocks, fees, nonces, receipts, and events.
- [Starknet.ID](/docs/hooks/use-stark-name): resolve Stark names, addresses, and profiles.

## Examples

- [Interactive demos](/demo): run the examples used by the reference pages.

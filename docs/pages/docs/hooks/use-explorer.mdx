# useExplorer

Access the current explorer, should be inside a `StarknetConfig`.

## Usage

```ts twoslash
import { useExplorer } from "@starknetfoundation/starknet-start-react";

const explorer = useExplorer();
```

## Returns

### explorer

- Type: `Explorer | null`

The current explorer, e.g. voyager. Returns `null` when the current chain
does not define an endpoint for the configured explorer (e.g. devnet), so
handle that case before building links.

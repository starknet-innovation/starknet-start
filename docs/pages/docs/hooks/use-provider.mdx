# useProvider

Hook for accessing the current provider.

## Usage

```ts twoslash
import { useProvider } from "@starknetfoundation/starknet-start-react";

const { provider, paymasterProvider } = useProvider();
```

## Returns

### provider

- Type: `ProviderInterface`

The current provider.

### paymasterProvider

- Type: `PaymasterInterface | undefined`

The current paymaster provider, if one is configured for the current chain.
